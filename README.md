
# BlackJack Game

A simple game of Blackjack created using python.

Below is a list showing the thought process behind creating the functions/methods  
required to create the game.



1. Shuffle cards 
2. let player place bet 
3. deal two cards to each person 
4. tell the player what cards he has 
5. tell the player one card of the dealer 
6. ask them if they want to hit or stay 
7. tell them if they bust 
8. if they don't bust, ask if they want to hit again

9. let the dealer play if the player stands
10. someone wins or loses 
11. pay them out 
12. ask if they want to play again
